# 个人博客 CMS 分期实施计划

> 本计划参考《个人博客 CMS 系统需求文档》的 Markdown 结构整理，用于把项目拆成可验收、可联调、可逐步扩展的多个阶段。

---

## 一、当前完成情况

### 第一期：前端初始化与苹果风后台

- Vite + Vue 3 + TypeScript 项目初始化
- Vue Router 路由配置
- Axios 请求封装
- Pinia 状态管理
- 后台登录页、后台布局、仪表盘基础可视化

### 第二期：登录与文章管理

- Spring Boot 3 后端工程初始化
- 管理员登录接口
- JWT 生成与前端登录状态保存
- 文章列表、新建、编辑、删除页面
- 文章模块从内存版升级为 MySQL + MyBatis 持久化

---

## 二、后续分期计划

### 第三期：分类与标签管理

#### 后端任务

- 分类表、标签表、文章标签关联表
- 分类 CRUD 接口
- 标签 CRUD 接口
- 文章创建/编辑时绑定分类和标签

#### 前端任务

- 分类管理页面
- 标签管理页面
- 文章编辑页加入分类选择和标签多选

#### 验收标准

- 能创建、编辑、删除分类和标签
- 文章能保存分类和标签关系
- 文章列表能展示分类和标签信息

---

### 第四期：前台博客展示

#### 后端任务

- 前台公开文章列表接口
- 前台文章详情接口
- 阅读量统计
- 分类/标签筛选接口

#### 前端任务

- 前台首页
- 文章详情页
- Markdown 渲染
- 分类/标签筛选

#### 验收标准

- 游客无需登录即可浏览已发布文章
- 文章详情能正确展示 Markdown 内容
- 草稿文章不会出现在前台

---

### 第五期：评论与图片上传

#### 后端任务

- 评论提交接口
- 评论审核接口
- 图片上传接口
- 本地上传目录静态访问

#### 前端任务

- 评论区组件
- 评论管理页面
- 文章封面图上传
- Markdown 编辑器图片插入

#### 验收标准

- 游客可提交评论
- 管理员可审核/删除评论
- 文章可设置封面图

---

### 第六期：仪表盘统计与部署优化

#### 后端任务

- 文章数量、访问量、评论数统计接口
- 最近文章、最近评论接口
- Swagger/OpenAPI 接口文档

#### 前端任务

- Dashboard 数据联调
- 响应式适配
- 构建与部署检查

#### 验收标准

- Dashboard 展示真实统计数据
- 前后台核心页面移动端可用
- 项目可通过 Nginx + JAR 方式部署
