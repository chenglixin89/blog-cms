## 三、运行说明

### 1. 前端运行

进入前端目录：

```bash
cd blog-frontend
```

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run dev
```

构建项目：

```bash
npm run build
```

### 2. 后端运行

进入后端目录，根据项目后端配置启动 Spring Boot 服务。前端 Axios 请求默认通过 `/api` 前缀访问后端接口。

## 五、答辩前检查

1. 确认前端依赖已安装。
2. 确认后端服务可以正常启动。
3. 确认浏览器可以访问首页。
4. 确认管理员账号可以登录后台。
5. 准备好 5 到 10 分钟演示流程。
6. 准备讲解 `src/api/http.ts` 和 `src/router/index.ts` 的核心代码。
7. 执行 `npm run build` 确认项目可以构建。
