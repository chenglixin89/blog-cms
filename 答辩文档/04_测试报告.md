## 五、接口测试说明

项目通过 Axios 请求后端接口，请求和响应均使用 JSON 格式。典型接口包括：

- `/front/articles`：前台文章列表
- `/front/articles/:id`：文章详情
- `/front/auth/login`：前台用户登录
- `/admin/articles`：后台文章管理
- `/admin/categories`：分类管理
- `/admin/tags`：标签管理
- `/admin/settings`：站点设置

## 六、异常测试

| 测试场景 | 预期处理 |
| --- | --- |
| 后端服务未启动 | 页面展示“无法连接到后端服务”提示 |
| Token 失效 | 清除本地登录状态并跳转登录页 |
| 请求超时 | 展示请求超时提示 |
| 必填项为空 | 页面阻止提交并提示用户 |
| JSON 路线图格式错误 | 站点设置页提示 JSON 格式不正确 |

## 七、构建测试

在 `blog-frontend` 目录执行：

```bash
npm run build
```

测试结果：构建通过，说明 TypeScript 类型检查和 Vite 打包流程正常。
