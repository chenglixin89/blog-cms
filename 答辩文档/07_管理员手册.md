# 个人博客 CMS 管理员手册

## 一、登录后台

1. 打开 `/login`。
2. 输入管理员账号和密码。
3. 登录成功后进入 `/admin/dashboard` 仪表盘。

后台页面需要管理员 Token 才能访问，未登录时会自动跳转到登录页。

## 二、仪表盘

仪表盘用于查看系统整体数据，包括：

- 文章总数
- 已发布文章数量
- 草稿数量
- 分类数量
- 标签数量
- 待审评论数量
- 文章状态分布
- 评论状态分布
- 最近 30 天发布趋势
- 热门文章互动指数

## 三、文章管理

进入“文章管理”后，管理员可以：

1. 查看文章列表。
2. 按关键词或状态筛选文章。
3. 新增文章。
4. 编辑已有文章。
5. 删除文章。
6. 预览文章。

文章编辑页支持标题、摘要、正文、封面、分类、标签、SEO 信息和 Markdown 内容编辑。

## 四、分类管理

分类用于组织文章栏目。管理员可以：

- 新增分类
- 修改分类名称和描述
- 调整分类排序
- 删除分类
- 查看分类下文章数量

## 五、标签管理

标签用于标记文章主题。管理员可以：

- 新增标签
- 设置标签颜色
- 修改标签排序
- 删除标签
- 查看标签关联文章数量

## 六、评论管理

评论管理用于维护文章评论区秩序。管理员可以：

- 搜索评论
- 审核评论
- 拒绝评论
- 回复评论
- 删除评论

## 七、留言管理

留言管理用于处理前台留言板内容。管理员可以：

- 查看留言列表
- 审核留言
- 回复留言
- 删除留言

## 八、用户管理

用户管理用于维护前台注册用户。管理员可以：

- 搜索用户
- 查看用户收藏、点赞和评论数量
- 启用用户
- 禁用用户
- 重置用户密码

## 九、媒体库

媒体库用于管理图片资源。管理员可以：

- 上传图片
- 按分类筛选图片
- 新增媒体分类
- 修改图片分类
- 复制图片地址
- 删除图片

## 十、Skills 管理

Skills 管理用于维护文章编辑相关扩展能力。管理员可以：

- 配置模型服务
- 测试连接
- 新增 Skill
- 编辑 Skill
- 启用或禁用 Skill
- 查看调用日志

## 十一、友链管理

友链管理用于维护友情链接。管理员可以：

- 审核友链申请
- 新增友链
- 编辑友链
- 删除友链
- 调整友链排序

## 十二、站点设置

站点设置用于维护前台展示信息，包括：

- 博客名称
- 副标题
- Logo
- 作者头像
- 首页主标题
- 首页简介
- 作者简介
- 每日一句 API
- 关于页路线图 JSON

## 十三、操作日志

操作日志记录管理员关键行为，方便追踪系统维护过程。管理员可以按关键词、模块、动作和分页条件查询日志。

## 十四、退出登录

点击后台顶部栏的退出按钮即可清除管理员 Token 并返回登录页。
